2.7 Таблицы символов Lisp парсера (Readtables)

Readtable является структурой данных, которая отображает символы в синтаксические типы для парсера Lisp выражений. В частности, readtable указывает для каждого строкового символа с синтаксисом макросимвола, какой макрос ему соответствует. Это механизм, с помощью которого пользователь может запрограммировать парсер для выполнения ограниченных, но полезных расширений. Смотрите раздел 22.1.5.
