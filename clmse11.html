2.9 Pathnames Имена файлов

Pathnames are the means by which a Common Lisp program can interface to an external ﬁle system in a reasonably implementation-independent manner. See section 23.1.1.

Имена файлов являеются сущностями, с помощью которых Common Lisp программа может взаимодействовать с внешней файловой системой в приемлемой платформонезависимой форме. Смотрите раздел 23.1.1.
