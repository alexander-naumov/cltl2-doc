2.11 Random-States Random-States

An object of type random-state is used to encapsulate state information used by the pseudo-random number generator. For more information about random-state objects, see section 12.9.

Объект типа random-state используется для инкапсулирования информации о состоянии, используемом генератором случайных чисел (ГСЧ). Для боле подробной информации об объектах random-state смотрите главу 12.9.
